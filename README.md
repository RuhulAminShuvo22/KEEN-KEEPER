🚀 KinKeeper
📌 Project Description
KinKeeper is a modern friendship tracking web application that helps users maintain meaningful relationships. Users can monitor when they last contacted friends, set communication goals, and log interactions like calls, texts, and video chats.
________________________________________
🛠️ Technologies Used
•	⚛️ React.js
•	🌐 React Router DOM
•	🎨 Tailwind CSS + DaisyUI
•	🔔 React Toastify
•	📊 Recharts
•	⚡ Vite
________________________________________
✨ Key Features
1️⃣ Friend Management System
•	View all friends in a clean card layout
•	Track days since last contact
•	Status indicators (Overdue / Almost Due / On Track)
2️⃣ Interaction Tracking (Call / Text / Video)
•	Log interactions instantly from Friend Details page
•	Timeline updates in real-time
•	Toast notifications for actions
3️⃣ Analytics Dashboard
•	Interactive Pie Chart using Recharts
•	Visual breakdown of Call / Text / Video interactions
•	Clean and responsive UI
________________________________________
📱 Responsive Design
•	Fully responsive across mobile, tablet, and desktop devices
________________________________________
⚠️ Extra Features
